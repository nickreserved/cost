Μεταβλητές
==========

Ονοματολογία
------------

$a, $aN, $b, $bN, $c, $cN, $d, $dN
	πρόχειρη μεταβλητή

$count, $countN
	μετρητής (Α/Α) σε πίνακα
	μετρητής υποφακέλων

$k, $kN
	κλειδί σε foreach

$v, $vN, $i
	τιμή σε foreach

$(filename)_*
	πρόχειρη μεταβλητή στο αρχείο filename.php
	εξαιρούνται οι δεσμευμένες μεταβλητές παρακάτω


Δεσμευμένες Μεταβλητές
----------------------

$data
	δεδομένα που εισάγονται από το πρόγραμμα

$bills
	array τιμολογίων
	συντόμευση στο $data['Τιμολόγια']

$bills_buy
	 array τιμολογίων ΜΗ παροχής υπηρεσιών
	 συντόμευση στο getBillsCategory($bills, 'Παροχή υπηρεσιών', false)

$bills_contract
	array τιμολογίων παροχής υπηρεσιών
	συντόμευση στο getBillsCategory($bills, 'Παροχή υπηρεσιών')

$bills_info
	array με πληροφορίες για το σύνολο των τιμολογίων
	συντόμευση στο calc_bills($bills)

$draft
	αν είναι true όσες διαταγές πάνε να εξαχθούν, εξάγονται σαν σχέδιο

$prereport
	αν είναι true η έκθεση βγαίνει σαν απαιτούμενη και όχι σαν γενόμενη

$onlyone
	αρχικοποιείται μόνο στην εξαγωγή δαπάνης
	αν θέλουμε να εξάγουμε μόνο ένα αντίγραφο για κάθε δικαιολογητικό είναι true

$bills_provider
	υπάρχει μεγάλη πιθανότητα να μην αρχικοποιηθεί
	array με κλειδί την «Επωνυμία» του προμηθευτή και τιμή ένα array με όλα τα τιμολόγια του συγκεκριμένου προμηθευτή
	συντόμευση στο bills_by_provider($bills)

$bills_month
	υπάρχει μεγάλη πιθανότητα να μην αρχικοποιηθεί
	array με κλειδί την «Επωνυμία» του προμηθευτή και τιμή ένα array με όλα τα τιμολόγια του συγκεκριμένου προμηθευτή
	συντόμευση στο bills_by_month(bills_with_fe($bills))
