Μεταβλητές
==========
Περιγραφή των μεταβλητών που υπάρχουν στη μηχανή προτύπων του php κώδικα.
Για την περίπτωση που κάποιος με γνώση php αποφασίσει να επεκτείνει ή τροποποιήσει το πρόγραμμα.
Για περισσότερη λεπτομέρεια, αφού δημιουργήσετε μια δαπάνη, βάλετε ένα var_dump σε κάθε μεταβλητή για να δείτε τι κλειδιά περιλαμβάνονται και τι περιεχόμενα λαμβάνουν.
Στο τέλος του κάθε δικαιολογητικού, εξαλείφονται με unset() οι μεταβλητές που ορίστηκαν μέσα στο δικαιολογητικό (σε περίπτωση που αυτές δε χρειάζεται να ξαναχρησιμοποιηθούν).

$data
	Δεδομένα που εισάγονται από το πρόγραμμα.
	Το init.php υπολογίζει και προσθέτει επιπλέον στοιχεία σε αυτά τα δεδομένα.

$init
	Τι αρχικοποίηση θα γίνει στο πρόγραμμα
	not set: Δεν έχει γίνει καμία αρχικοποίηση
	1: Βασική αρχικοποίηση για το PHP script
	2: Προσαρμοσμένος Error Handler για τα PHP scripts
	3: unserialize του stdin stream
	4: Στοιχεία ώστε να έχουμε αν κοινοποιείται η δαπάνη στο διαδίκτυο
	5: Στοιχεία τιμολογίων
	6: Στοιχεία συμβάσεων
	7: Στοιχεία διαγωνισμών

$output
	Αν είναι false όσες διαταγές πάνε να εξαχθούν, εξάγονται σαν σχέδιο.
	Αν είναι true όσες διαταγές πάνε να εξαχθούν, εξάγονται σαν ακριβές αντίγραφο.
	Αν είναι 'δαπάνη' εξάγεται πλήρης δαπάνη και όσες διαταγές πάνε να εξαχθούν, εξάγονται σαν ακριβές αντίγραφο.

$onlyone
	Αρχικοποιείται μόνο στην εξαγωγή δαπάνης.
	Αν θέλουμε να εξάγουμε μόνο ένα αντίγραφο για κάθε δικαιολογητικό είναι true.

$paper_v
	Αρχικοποιείται μόνο στην εξαγωγή δαπάνης.
	Απαριθμεί ένα-ένα τα δικαιολογητικά του φύλλου καταχώρησης:
	foreach($data['Φύλλο Καταχώρησης'] as $paper_v) ...
