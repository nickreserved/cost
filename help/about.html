Τί κάνει το πρόγραμμα

Το πρόγραμμα «Στρατιωτικές Δαπάνες» δημιουργήθηκε προκειμένου να συντάσσονται αυτόματα όλα τα δικαιολογητικά των δαπανών, να αποφεύγονται τα λάθη και να μειώνεται ο χρόνος συντάξεως της δαπάνης.

Τα εξαγόμενα δικαιολογητικά σώζονται από το πρόγραμμα σε αρχεία κειμένου, τα οποία ανοίγουν από οποιοδήποτε επεξεργαστή κειμένου (για εκτύπωση ή περεταίρω επεξεργασία).

Το ρεκόρ χρόνου που έχω κάνει για σύνταξη δαπάνης ήταν 2 λεπτά.

Το πρόγραμμα υποστηρίζει κοινές δαπάνες, δαπάνες έργων και δαπάνες με διαγωνισμούς. Ωστόσο, επειδή για τις κατασκευές έργων με δημόσιο διαγωνισμό την ευθύνη έχουν οι ΔΣΕ, το πρόγραμμα δεν υποστηρίζει δαπάνες έργων με δημόσιο διαγωνισμό.

Περί του προγραμματιστή

Ονομάζομαι Γκέσος Παύλος (ΣΣΕ 2002), γεννήθηκα στην Λάρισα το 1980 αλλά λόγω δουλειάς τώρα πια δεν έχω ιδιαίτερη πατρίδα.

Ασχολούμαι με προγραμματισμό από το 1994 όταν είδα για πρώτη φορά στη ζωή μου υπολογιστή.

Γνωρίζω άριστα C++, C, Java (εφαρμογές, applet, κινητά), Web Design (με PHP, MySQL, AJAX και όλα τα W3C Specifications).

Στο παρελθόν έχω ασχοληθεί (κατά σειρά εξοικείωσης) με x86 Assembly, Quick Basic, Visual Basic, Delphi.

Έχω δουλέψει με πάρα πολλές βιβλιοθήκες και APIs, κυρίως για C και C++ οι κυριότερες των οποίων είναι (κατά σειρά εξοικείωσης) οι OpenGL, boost, Qt, SDL, Windows API, TinyXML, OpenAL, wxWidgets, βιβλιοθήκες πολυμέσων (εικόνων, ήχου, video, multimedia container), βιβλιοθήκες συμπίεσης αρχείων, freetype.

Έχω μελετήσει τα specifications πάρα πολλών τύπων αρχείων καθώς και πολλά RFCs.

Έχω στήσει και συντηρήσει σχεδόν όλους τους κλασικούς servers.

Σήμερα προγραμματίζω κυρίως cross-platform, open-source κώδικα.

Μελλοντικά ενδιαφέρομαι να προγραμματίζω (κατά σειρά προτεραιότητας) για Linux καθώς και drivers για συσκευές USB.

Απαιτήσεις του προγράμματος

  • Λειτουργικό σύστημα: Όλα όσα υποστηρίζονται από τη Java και το PHP. Μερικά από αυτά είναι τα Windows, το Linux, το MacOS.
  • Επεξεργαστής Κειμένου: Όλοι όσοι υποστηρίζουν το RTF Specification 1.6. Μερικοί είναι το OpenOffice, το Microsoft Word.
  • Υπολογιστής: Οι υπολογιστές από το 2001 και μετά θεωρούνται επαρκείς.
  • Διάφορα πακέτα:
    • Java: Απαιτείται η εγκατάσταση του Java Runtime Environment (jre) 1.5 και πάνω.
    • PHP: Απαιτείται η εγκατάσταση του PHP 5 και πάνω.
