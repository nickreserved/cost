Τα php scripts αυτά μετατρέπουν τα αρχεία providers.ini και men.ini από το Cost 0.3.1 στο Cost 1.0.0.
Πρέπει να τα τρέξετε από το directory του Cost 0.3.1 περίπου έτσι:

php -n provider.php >out.txt

Το αρχείο out.txt έχει δεδομένα που πρέπει να εισαχθούν με προσοχή στο main.ini του Cost 1.0.0.
Αν δεν ξέρετε ακριβώς πως να το κάνετε, μπορεί να καταστρέψετε το αρχείο main.ini του Cost 1.0.0 και να χρειαστεί να καταχωρήσετε τα δεδομένα του από την αρχή.
