Τί κάνει το πρόγραμμα

Το πρόγραμμα «Στρατιωτικές Δαπάνες» δημιουργήθηκε προκειμένου να συντάσσονται αυτόματα όλα τα δικαιολογητικά των δαπανών, να αποφεύγονται τα λάθη και να μειώνεται ο χρόνος συντάξεως της δαπάνης.

Τα εξαγόμενα δικαιολογητικά σώζονται από το πρόγραμμα σε αρχεία κειμένου, τα οποία ανοίγουν από οποιοδήποτε επεξεργαστή κειμένου (για εκτύπωση ή περαιτέρω επεξεργασία).

Το ρεκόρ χρόνου που έχω κάνει για σύνταξη της πιο απλής δαπάνης ήταν 2 λεπτά.

Το πρόγραμμα υποστηρίζει δαπάνες με απευθείας ανάθεση, με συνοπτικούς διαγωνισμούς ή διαγωνισμούς με ανοικτή και κλειστή διαδικασία και αντίστοιχα με έργα. Δαπάνες με διαγωνισμούς ανοικτής και κλειστής διαδικασίας καθώς και δαπάνες έργων, λόγω της εν εξελίξει διαδικασίας προσαρμογής στο ΤΕ 61-4/2018/ΓΕΣ/ΔΟΙ, δεν υποστηρίζονται ακόμα.

Το πρόγραμμα ξεκίνησε να προγραμματίζεται τον Ιούλιο του 2004.

Περί του προγραμματιστή

Ονομάζομαι Γκέσος Παύλος (ΣΣΕ 2002), γεννήθηκα στην Λάρισα το 1980, αλλά λόγω δουλειάς τώρα πια δεν έχω ιδιαίτερη πατρίδα.

Ασχολούμαι με προγραμματισμό από το 1994 όταν είδα για πρώτη φορά στη ζωή μου υπολογιστή.

Γνωρίζω άριστα C++, Java, Web Design (PHP, MySQL, Javascript και όλα τα W3C Specifications), OpenGL και 3D Modeling & Animation.

Έχω δουλέψει με πάρα πολλές βιβλιοθήκες και APIs, κυρίως για C και C++, οι κυριότερες των οποίων είναι (κατά σειρά εξοικείωσης) οι OpenGL, boost, SDL, Windows API, TinyXML, OpenAL, βιβλιοθήκες πολυμέσων (εικόνων, ήχου, video, multimedia container), βιβλιοθήκες συμπίεσης αρχείων, freetype, wxWidgets.

Έχω μελετήσει τα specifications πάρα πολλών τύπων αρχείων καθώς και πολλά RFCs.

Έχω στήσει και συντηρήσει σχεδόν όλους τους κλασικούς servers.

Πριν πολλά χρόνια είχα ασχοληθεί (κατά σειρά εξοικείωσης) με x86 Assembly, Quick Basic, Qt, Visual Basic, Delphi.

Σήμερα προγραμματίζω κυρίως cross-platform, open-source κώδικα.

Απαιτήσεις του προγράμματος

  • Λειτουργικό σύστημα: Όλα όσα υποστηρίζονται από τη Java 1.8 και το PHP 5.3. Μερικά από αυτά είναι τα Windows XP SP3, το Linux, το Mac OS X.
  • Επεξεργαστής Κειμένου: Όλοι όσοι υποστηρίζουν το RTF Specification 1.6. Μερικοί είναι το LibreOffice, το Microsoft Word.
  • Υπολογιστής: Οι υπολογιστές από το 2008 και μετά θεωρούνται επαρκείς.
  • Διάφορα πακέτα: Για την λειτουργία του προγράμματος απαιτείται κατ' ελάχιστον Java 8 (1.8) και PHP 5.3. Η εγκατάσταση του προγράμματος για τα Windows εγκαθιστά αυτόματα και τα παραπάνω πακέτα, οπότε δε χρειάζονται παραπάνω ενέργειες.
  • Προειδοποίηση: Απροσδιόριστη στιγμή μετά την 1 Ιαν 2021, το πρόγραμμα θα απαιτεί κατ' ελάχιστον Java 11. Αυτό σημαίνει ότι θα πρέπει μέχρι τότε να έχετε εξασφαλίσει τουλάχιστον Windows 7 64 bit. Υπολογιστές με Windows XP ή λειτουργικό 32 bit δεν θα υποστηρίζονται.
